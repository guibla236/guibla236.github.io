Dashboard de tiendas

Concepto
Permite extraer todos las publicaciones de Mercado Libre asociadas a una búsqueda y decidir las mejores oportunidades.
Gráfico
Para ello, grafica precios y cantidad de ventas.
Ajustes
Permite filtrar los datos de acuerdo a sus características, precio y/o vendedor

Tecnologías usadas

ReactJS
Para el procesamiento de los datos extraídos y los gráficos
Bootstrap
Diseño de interfaces
Flask
Backend del extractor de artículos
JQuery
Frontend del extractor de artículos
